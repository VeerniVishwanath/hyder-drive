## HyderDrive: Premium Car Rentals
Welcome to HyderDrive a car rental service from Hyderabad built using NextJS.

Live at [here](https://hyder-drive.vercel.app/)

## Technologies used
**Framework** : NextJS \
**UI Library** : Shadcn \
**Formatter and Linter** : Biome
